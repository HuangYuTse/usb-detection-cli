# usb-detection-cli
![logo of usb-detection-cli](logo.png)
usb detecter by CLI interface 

這是一個用node.js寫的CLI介面USB裝置偵測與分析，目前還在增加功能中敬請期待
## Version
beta
v0.2.0
## Getting started

```bash
git clone https://github.com/HuangYuTse/usb-detection-cli.git 
```
```bash
npm install
```
```bash
npm start
```
## Dependencies packages
